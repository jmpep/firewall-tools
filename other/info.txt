A practical solution is usually split into 3 scripts (or one script with subcommands):

list_packages.py
Login to Management Server
List all policy packages
List access policies and layers
download_package.py
Download a specific package
Download all packages
Export to JSON
policy_objects.py
Download a policy package
Extract all UIDs referenced by:
Access rules
NAT rules
Threat rules
Inline layers
Groups
Download all corresponding objects
Produce:
policy.json
objects.json
Verify every UID in policy exists in objects
